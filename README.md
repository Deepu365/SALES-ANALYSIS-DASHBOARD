# SALES-ANALYSIS-DASHBOARD
A data-drive"Dashboard that analyzes sales performance and visualizes key trends to support better business decisions."
SALES-ANALYSIS-DASHBOARD

Description:
this project analyzes sales data using python. it reads a csv file, calculates total and daily sales, and shows sales trend using a graph.

Technologies: 
python pandas matplotlib

Features: 
load sales data from csv calculate total sales group sales by date show daily sales show sales trend graph

output: 
data printed in terminal daily sales result line graph of sales trend

How to run:

install libraries pip install pandas matplotlib keep sales.csv in same folder run program python main.py
